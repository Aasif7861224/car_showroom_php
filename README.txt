SETUP
1) Copy folder to: C:\xampp\htdocs\car_showroom_php\
2) Import database:
   phpMyAdmin -> Import -> database/database.sql
3) Configure DB:
   app/config.php (host/port/user/pass)
4) Run:
   http://localhost/car_showroom_php/
